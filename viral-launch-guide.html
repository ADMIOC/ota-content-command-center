How to use Immediate Posting Readiness

Tutorial Section Open Launch Control

No-brainer operating map

Use this panel to decide what can be posted now, what is missing, and what the publisher needs next.

Viral Launch Control is the final pre-posting checkpoint. It does not publish automatically. It turns your campaign state into one visible action path: pick the candidate, run the Launch Agent, review queued agent work, prepare manual readiness, and copy the launch packet.

Launch score 0-57

Build the missing campaign inputs before posting.

58-77

Review final gaps and decide whether a human can proceed.

78-100

Ready for a controlled manual launch when approvals are clear.

Read the scoreboard

What each top card tells you

Launch score

Measures hook strength and launch inputs. Low score means build, not publish.

Checklist

Shows how many launch gates are clear. Treat every OPEN item as an agent action queue.

Candidate

Confirms whether a selected scene exists. Zero means go create the first scripted scene.

Compliance

Clear means standard manual review is enough. Hold means approvals must be completed first.

The eight launch gates

Use the checklist as your tactical work queue

01 Hook opens with tension

The Launch Agent strengthens the first line with a problem, contrast, urgency, risk, or before-and-after shift.

Run Gate Agent
02 Script can travel to audio

The Launch Agent drafts script text when the selected candidate is missing a voiceover spine.

Run Gate Agent
03 Visual prompt supports the story

The Launch Agent drafts the prompt around script, emotional stakes, and cinematic brand perspective.

Run Gate Agent
04 Audio or Remotion source exists

The Launch Agent queues ElevenLabs narration and makes the pending audio handoff visible.

Queue Audio Agent
05 Publishable media exists

The Launch Agent queues Remotion/Higgsfield media production and tracks the required returned output.

Queue Media Agent
06 Caption package is ready

The Launch Agent drafts caption, hashtags, and platform notes from the selected hook.

Run Caption Agent
07 Calendar slot is ready

The Launch Agent creates or marks the primary calendar slot as Ready.

Run Calendar Agent
08 Human review path is visible

The Launch Agent confirms standard review or queues regulated approval review.

Run Approval Agent

Decision rule

When do I click each button?

Run Scoring Agent

Click after any campaign input changes. It refreshes the score and the launch gates.

Agent Prepare Manual Launch

Click when the agent has prepared the launch packet and the human operator accepts any remaining risk.

Agent Copy Launch Pack

Click when the publisher needs the final posting packet, including warnings and manual posting steps.

What the publisher receives

The Viral Launch Pack removes guessing

The copied packet includes the campaign, brand, platform targets, selected scene, opening hook, script, prompt, publishing package, readiness checklist, compliance warning, and manual posting steps.
